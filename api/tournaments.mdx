## Create Tournament

Create a new arena tournament.

```bash
POST /api/tournament
```

Requires `tournament:write` OAuth scope.

<ParamField body="name" type="string" required>
  Tournament name (3-40 characters)
</ParamField>

<ParamField body="clockTime" type="number" required>
  Clock initial time in minutes (0-180)
</ParamField>

<ParamField body="clockIncrement" type="integer" required>
  Clock increment in seconds (0-180)
</ParamField>

<ParamField body="minutes" type="integer" required>
  Tournament duration in minutes (20-720)
</ParamField>

<ParamField body="startDate" type="timestamp">
  Tournament start date (defaults to now + 10 minutes)
</ParamField>

<ParamField body="variant" type="string" default="standard">
  Chess variant
</ParamField>

<ParamField body="rated" type="boolean" default="true">
  Whether games are rated
</ParamField>

<ParamField body="position" type="string">
  Custom starting position in FEN
</ParamField>

<ParamField body="berserkable" type="boolean" default="true">
  Allow berserk
</ParamField>

<ParamField body="streakable" type="boolean" default="true">
  Allow arena streaks
</ParamField>

<ParamField body="hasChat" type="boolean" default="true">
  Enable tournament chat
</ParamField>

<ParamField body="description" type="string">
  Tournament description
</ParamField>

<ParamField body="password" type="string">
  Password for private tournament
</ParamField>

<ParamField body="teamId" type="string">
  Restrict to team members
</ParamField>

<ParamField body="minRating" type="integer">
  Minimum rating requirement
</ParamField>

<ParamField body="maxRating" type="integer">
  Maximum rating requirement
</ParamField>

<ParamField body="nbRatedGame" type="integer">
  Minimum number of rated games required
</ParamField>

<CodeGroup>
```bash cURL
curl -X POST https://lichess.org/api/tournament \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "name=My Arena" \
  -d "clockTime=3" \
  -d "clockIncrement=2" \
  -d "minutes=45" \
  -d "rated=true"
```

```bash Team Battle
curl -X POST https://lichess.org/api/tournament \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "name=Team Battle" \
  -d "clockTime=5" \
  -d "clockIncrement=3" \
  -d "minutes=120" \
  -d "teamBattleByTeam=team1,team2,team3"
```
</CodeGroup>

<ResponseExample>
```json
{
  "id": "newtourn1",
  "fullName": "My Arena",
  "url": "https://lichess.org/tournament/newtourn1"
}
```
</ResponseExample>

## Export Tournament Games

Download all games from a tournament.

```bash
GET /api/tournament/{id}/games
```

<ParamField path="id" type="string" required>
  Tournament ID
</ParamField>

<ParamField query="player" type="string">
  Filter by username
</ParamField>

<ParamField query="moves" type="boolean" default="true">
  Include moves
</ParamField>

<ParamField query="pgnInJson" type="boolean" default="false">
  Include PGN in JSON responses
</ParamField>

<ParamField query="tags" type="boolean" default="true">
  Include PGN tags
</ParamField>

<ParamField query="clocks" type="boolean" default="false">
  Include clock times
</ParamField>

<ParamField query="evals" type="boolean" default="false">
  Include analysis evaluations
</ParamField>

<ParamField query="opening" type="boolean" default="false">
  Include opening information
</ParamField>

<CodeGroup>
```bash PGN Format
curl "https://lichess.org/api/tournament/abc123/games" \
  -H "Accept: application/x-chess-pgn"
```

```bash NDJSON Format
curl "https://lichess.org/api/tournament/abc123/games?player=username"
```
</CodeGroup>

Returns games in streaming PGN or NDJSON format.

## Export Tournament Results

Export tournament results and standings.

```bash
GET /api/tournament/{id}/results
```

<ParamField path="id" type="string" required>
  Tournament ID
</ParamField>

<ParamField query="nb" type="integer">
  Maximum number of players to fetch
</ParamField>

<ParamField query="sheet" type="boolean" default="false">
  Include score sheet for each player (slower)
</ParamField>

<ParamField query="as" type="string">
  Format: `json` or `csv`
</ParamField>

<CodeGroup>
```bash NDJSON (default)
curl https://lichess.org/api/tournament/abc123/results
```

```bash CSV Format
curl "https://lichess.org/api/tournament/abc123/results?as=csv"
```

```bash With Score Sheets
curl "https://lichess.org/api/tournament/abc123/results?sheet=true"
```
</CodeGroup>

<ResponseExample>
```json NDJSON Line
{
  "rank": 1,
  "score": 42,
  "rating": 2800,
  "username": "player1",
  "title": "GM",
  "performance": 2850,
  "team": "team-id",
  "sheet": {
    "scores": "3 2 3 3 2 3",
    "total": 16,
    "fire": true
  }
}
```
</ResponseExample>

## Swiss Tournaments

### Create Swiss Tournament

```bash
POST /api/swiss/new/{teamId}
```

<ParamField path="teamId" type="string" required>
  Team ID (Swiss tournaments require a team)
</ParamField>

Accepts similar parameters to arena tournaments plus:

<ParamField body="nbRounds" type="integer" required>
  Number of rounds (3-100)
</ParamField>

<ParamField body="roundInterval" type="integer">
  Seconds between rounds (5-99999, or 99999999 for manual)
</ParamField>

### Get Swiss Tournament

```bash
GET /api/swiss/{id}
```

### Join Swiss Tournament

```bash
POST /api/swiss/{id}/join
```

### Withdraw from Swiss

```bash
POST /api/swiss/{id}/withdraw
```

### Export Swiss Games

```bash
GET /api/swiss/{id}/games
```

### Export Swiss Results

```bash
GET /api/swiss/{id}/results
```

### Manually Schedule Next Round

```bash
POST /api/swiss/{id}/schedule-next-round
```

<ParamField body="date" type="timestamp">
  When to start the next round (default: now)
</ParamField>

### Get Team Swiss Tournaments

```bash
GET /api/team/{teamId}/swiss
```
